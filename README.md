# Yudnaris Prado | Portafolio Profesional

Portafolio web interactivo que muestra mi trayectoria como Analista BackOffice, Especialista Cloud y Quality Control. Diseñado para destacar mis habilidades técnicas, experiencia profesional y proyectos académicos.

## 🌟 Características

- **Diseño Responsive** - Adaptado para todos los dispositivos (móvil, tablet, escritorio)
- **Modo Claro/Oscuro** - Alterna entre temas visuales con persistencia en localStorage
- **Bilingüe (Español/Inglés)** - Cambio de idioma instantáneo sin recargar la página
- **Código QR Dinámico** - Comparte el portafolio fácilmente desde cualquier dispositivo
- **Carrusel de Testimonios** - Opiniones de colegas, profesores y compañeros
- **Sección de Proyectos** - Enlaces directos a repositorios de GitHub
- **Contacto Directo** - Enlaces a LinkedIn, GitHub, email y WhatsApp

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura del sitio |
| CSS3 | Estilos personalizados y temas claro/oscuro |
| Bootstrap 5 | Framework CSS responsive |
| JavaScript | Funcionalidades interactivas (idioma, tema, QR) |
| Font Awesome 6 | Iconos vectoriales |
| QRCode.js | Generación de código QR dinámico |
| Google Fonts | Tipografías Playfair Display y DM Sans |
